# توثيق مهام Cron التلقائية (Vercel Cron Jobs)

## 📋 نظرة عامة

يحتوي هذا الملف على إعدادات مهام Cron التلقائية المجدولة على Vercel لتنفيذ عمليات الصيانة الدورية في منصة Marketna.

**الملف:** `vercel.json`

---

## 🕐 صيغة Cron Expression

تستخدم Vercel صيغة Cron القياسية المكونة من 5 خانات:

```
┌───────────── الدقيقة (0 - 59)
│ ┌───────────── الساعة (0 - 23) بتوقيت UTC
│ │ ┌───────────── يوم الشهر (1 - 31)
│ │ │ ┌───────────── الشهر (1 - 12)
│ │ │ │ ┌───────────── يوم الأسبوع (0 - 6) حيث 0 = الأحد
│ │ │ │ │
* * * * *
```

### وصف الخانات

| الخانة | المجال | القيم | الوصف |
|--------|--------|-------|-------|
| 1 | الدقيقة | 0-59 | دقيقة التنفيذ |
| 2 | الساعة | 0-23 | ساعة التنفيذ (UTC) |
| 3 | يوم الشهر | 1-31 | يوم الشهر |
| 4 | الشهر | 1-12 | الشهر |
| 5 | يوم الأسبوع | 0-6 | يوم الأسبوع (0 = الأحد) |

## 📅 المهام المُعدّة

### 1. مهمة تنظيف الرموز (`cleanup-tokens`)

**المسار:** `/api/cron/cleanup-tokens`  
**الجدول:** `0 2 * * *`

#### الجدول الزمني

| المنطقة الزمنية | الوقت |
|-----------------|-------|
| UTC | 02:00 صباحاً |
| سوريا (UTC+3) | 05:00 صباحاً |
| السعودية (UTC+3) | 05:00 صباحاً |
| الإمارات (UTC+4) | 06:00 صباحاً |

#### الوصف

تقوم هذه المهمة بحذف رموز إعادة تعيين كلمة المرور القديمة والمنتهية الصلاحية من قاعدة البيانات.

**معايير الحذف:**
- رموز منتهية منذ أكثر من 30 يوم
- رموز مُستخدمة منذ أكثر من 7 أيام

---

### 2. مهمة تحديث الأسعار (`update-rates`)

**المسار:** `/api/cron/update-rates`  
**الجدول:** `10 2 * * *`

#### الجدول الزمني

| المنطقة الزمنية | الوقت |
|-----------------|-------|
| UTC | 02:10 صباحاً |
| سوريا (UTC+3) | 05:10 صباحاً |
| السعودية (UTC+3) | 05:10 صباحاً |
| الإمارات (UTC+4) | 06:10 صباحاً |

#### الوصف

تقوم هذه المهمة بتحديث أسعار العملات من مصدر خارجي.

**ملاحظة:** تم ضبطها بعد 10 دقائق من مهمة التنظيف لضمان اكتمال العمليات الأولى.

## 📋 أمثلة شائعة على Cron Expressions

### مهام يومية

| التعبير | UTC | سوريا (UTC+3) | الوصف |
|---------|-----|---------------|-------|
| `0 0 * * *` | 00:00 | 03:00 | يومياً منتصف الليل |
| `0 2 * * *` | 02:00 | 05:00 | يومياً الساعة 2 صباحاً |
| `0 6 * * *` | 06:00 | 09:00 | يومياً الساعة 6 صباحاً |
| `0 12 * * *` | 12:00 | 15:00 | يومياً ظهراً |
| `0 18 * * *` | 18:00 | 21:00 | يومياً مساءً |
| `0 23 * * *` | 23:00 | 02:00+1 | يومياً قبل منتصف الليل |

### مهام متكررة

| التعبير | الوصف |
|---------|-------|
| `*/5 * * * *` | كل 5 دقائق |
| `*/10 * * * *` | كل 10 دقائق |
| `*/15 * * * *` | كل 15 دقيقة |
| `*/30 * * * *` | كل 30 دقيقة |
| `0 */2 * * *` | كل ساعتين (عند الدقيقة 0) |
| `0 */4 * * *` | كل 4 ساعات |
| `0 */6 * * *` | كل 6 ساعات |
| `0 */12 * * *` | كل 12 ساعة |

### مهام أسبوعية

| التعبير | UTC | الوصف |
|---------|-----|-------|
| `0 0 * * 0` | 00:00 | كل أحد الساعة 00:00 |
| `0 0 * * 1` | 00:00 | كل اثنين الساعة 00:00 |
| `0 0 * * 5` | 00:00 | كل جمعة الساعة 00:00 |
| `0 0 * * 6` | 00:00 | كل سبت الساعة 00:00 |

### مهام شهرية

| التعبير | UTC | الوصف |
|---------|-----|-------|
| `0 0 1 * *` | 00:00 | أول كل شهر |
| `0 0 15 * *` | 00:00 | منتصف كل شهر |
| `0 0 1 1 *` | 00:00 | أول يناير (سنوي) |

## 🌍 تحويل التوقيت

### المناطق الزمنية الشائعة

| المنطقة | UTC Offset | مثال (UTC 02:00) |
|---------|------------|------------------|
| UTC | +0 | 02:00 |
| سوريا | +3 | 05:00 |
| السعودية | +3 | 05:00 |
| مصر | +2 | 04:00 |
| الإمارات | +4 | 06:00 |
| الكويت | +3 | 05:00 |
| قطر | +3 | 05:00 |
| البحرين | +3 | 05:00 |
| عمان | +4 | 06:00 |
| الأردن | +3 | 05:00 |
| لبنان | +3 | 05:00 |
| العراق | +3 | 05:00 |

### قاعدة التحويل

```
التوقيت المحلي = UTC + Offset المنطقة
```

**مثال:**
```
UTC: 02:00
سوريا (UTC+3): 02:00 + 3 = 05:00
الإمارات (UTC+4): 02:00 + 4 = 06:00
```

## 🔧 كيفية التعديل

### 1. تحديد الوقت المطلوب

```
مثال: أريد المهمة الساعة 03:30 فجراً بتوقيت سوريا

الخطوات:
1. سوريا UTC+3
2. 03:30 سوريا = 00:30 UTC
3. التعبير: "30 0 * * *"
```

### 2. تحديث ملف vercel.json

```json
{
  "crons": [
    {
      "path": "/api/cron/cleanup-tokens",
      "schedule": "30 0 * * *"
    }
  ]
}
```

### 3. النشر على Vercel

```bash
git add vercel.json
git commit -m "Update cron schedule"
git push
```

### مولدات Cron Online

- [crontab.guru](https://crontab.guru/) - محرر تفاعلي

### مكتبات JavaScript

```typescript
// cron-parser
import { parseExpression } from 'cron-parser'

const interval = parseExpression('0 2 * * *')
console.log(interval.next().toString())
// Next execution time

// node-cron
import cron from 'node-cron'

cron.schedule('0 2 * * *', () => {
  console.log('Running cron job')
})
```
