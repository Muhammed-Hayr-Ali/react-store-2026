# 🌱 بيانات أولية (Seed Data)

## 📋 محتويات المجلد

هذا المجلد يحتوي البيانات الأساسية اللازمة لتشغيل النظام لأول مرة.

| الملف                | الوصف                                   | الحالة   |
| -------------------- | --------------------------------------- | -------- |
| `001_role_seed.sql`  | إدخال الأدوار والصلاحيات (5 أدوار)      | ✅ مطلوب |
| `002_store_seed.sql` | إعدادات المتجر الافتراضية + أسعار الصرف | ✅ مطلوب |

---

## 🚀 كيفية الاستخدام

### 1️⃣ تشغيل ملف الأدوار

```sql
-- 004_Seed Data/001_role_seed.sql
-- يقوم بإدخال 5 أدوار رئيسية مع صلاحياتها
```

**الأدوار التي سيتم إضافتها:**

1.  👑 **admin**: مدير النظام (صلاحيات كاملة).
2.  👨‍💼 **vendor**: موظف/بائع (إدارة المنتجات والطلبات).
3.  👤 **customer**: عميل (تصفح وشراء).
4.  🚚 **delivery**: موظف توصيل (تسليم الطلبات عبر QR).
5.  🎧 **support**: دعم فني (إدارة التذاكر).

### 2️⃣ تشغيل ملف إعدادات المتجر والعملات

```sql
-- 004_Seed Data/002_store_seed.sql
-- يقوم بإدخال إعدادات المتجر الافتراضية + 8 عملات
```

**البيانات التي سيتم إضافتها:**

1.  🏪 **إعدادات المتجر**: اسم المتجر، الوصف، معلومات الاتصال.
2.  💱 **أسعار الصرف**: 8 عملات (USD, SAR, AED, EGP, KWD, SYP, EUR, GBP).

### ⚠️ ملاحظات هامة

- ✅ **ترتيب التنفيذ**: يجب تشغيل هذه الملفات **بعد** `003_RLS Policies` و**قبل** `005_Trigger Functions`.
- ✅ **آمن للتكرار**: يستخدم `ON CONFLICT DO UPDATE`، لذا يمكن تشغيله عدة مرات بأمان.
- 🔄 **تحديث أسعار الصرف**: يتم تحديث أسعار الصرف تلقائياً عبر Cron Job.
