# قالب فارغ (Empty Schema Template)

## 📋 نظرة عامة

قالب فارغ لإنشاء جداول جديدة في منصة Marketna للتجارة الإلكترونية.

**الإصدار:** 1.0  
**التاريخ:** 2026-03-21  
**الاعتماديات:** حسب الحاجة

---

## 📁 محتويات المجلد

| الملف                 | الوصف                    |
| --------------------- | ------------------------ |
| `create_table.sql`    | قالب إنشاء جدول جديد     |
| `create_function.sql` | قالب الدوال              |
| `create_policy.sql`   | قالب سياسات الأمان       |
| `create_data.sql`     | قالب البيانات الافتراضية |

---

## 📝 طريقة الاستخدام

1. انسخ هذا المجلد وسمّه باسم الجدول الجديد
2. عدّل ملفات SQL حسب الحاجة
3. اتبع التنسيق الموحد للملفات

---

## 📊 هيكلية الملفات الموصى بها

```sql
-- create_table.sql
-- 1. تنظيف كامل قبل الإنشاء
-- 2. إنشاء الجدول
-- 3. الفهارس
-- 4. التعليقات

-- create_function.sql
-- 1. دالة 1
-- 2. دالة 2
-- ...

-- create_policy.sql
-- 1. تفعيل RLS
-- 2. سياسات القراءة
-- 3. سياسات الكتابة

-- create_data.sql
-- 1. البيانات الافتراضية
```
