# 003_RLS Policies — سياسات أمان مستوى الصفوف

> **الوصف:** يحتوي على جميع سياسات Row Level Security (RLS) التي تحمي بيانات كل جدول بناءً على دور المستخدم.

---

## 📂 محتويات المجلد

| الملف | الوصف | ترتيب التنفيذ |
|-------|-------|:---:|
| [`002_rls_policies.sql`](./002_rls_policies.sql) | 70+ سياسة RLS + Column-Level Security | **الخطوة 3** |

---

## 📋 ما يفعله هذا الملف

- إعادة تعيين صلاحيات الأعمدة على `core_profile` (Column-Level Security)
- تفعيل RLS على جميع الجداول الـ 23
- إنشاء 70+ سياسة وصول (Policy)

---

## 🔐 مستويات الوصول

| المستوى | الوصف | أمثلة |
|---------|-------|-------|
| **PUBLIC** | أي شخص (بدون تسجيل) | عرض المنتجات النشطة، المتاجر، التقييمات |
| **authenticated** | المستخدم المسجل | إدارة البروفايل، الطلبات، المفضلة |
| **admin** | مدير النظام | إدارة كل شيء — يتجاوز جميع القيود |
